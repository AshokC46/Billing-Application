# Billing Application

A **React + Redux Toolkit + TypeScript** web application that simulates a shopping cart and billing system.  

The app allows users to:

- Add products to cart
- Increase or decrease product quantity
- Remove products from cart
- View **subtotal**, **special offers**, **total savings**, and **final total**
- See special offers in a **collapsible accordion**  

The application uses **Redux Toolkit** for state management and is fully compatible with **Redux DevTools** for easy state debugging.  

---

## **Features**

- **Product List**: Bread, Milk, Butter, Chocolates, Ice Cream, Cool Drinks, Cheese  
- **Cart Sidebar**:  
  - Shows all added products with quantity controls  
  - Dynamically updates subtotal and totals  
  - Collapsible **Special Offers** section with details  
- **Special Offers** (examples):
  - Buy 1 Cheese → Get 1 Cheese Free  
  - Buy Milk → Get Bread at Half Price  
  - 10% off on orders above ₹500  
- **Checkout button** (dummy, for UI demonstration)  
- Fully responsive and interactive UI using **Bootstrap 5**  

---

## **Technologies Used**

- **React 18**  
- **Redux Toolkit**  
- **TypeScript**  
- **Bootstrap 5** for styling  
- **Vite** for development and build  
- **Netlify** for deployment
